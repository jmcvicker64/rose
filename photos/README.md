# Photos 🌹

These `pXXXX.jpg` files are **auto-generated** — web-sized copies of your library,
numbered in date order. The app chooses each day's picture by capture date (see the
main [`../README.md`](../README.md)).

- [`_source-map.tsv`](_source-map.tsv) maps each `pXXXX.jpg` → its capture date and
  original filename, so you can tell which photo is which.
- To remove one, delete the `pXXXX.jpg` here **and** its line in [`../photos.js`](../photos.js).
- To add more or rebuild, ask Claude to re-run the photo build.
